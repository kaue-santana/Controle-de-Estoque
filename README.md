# Controle de Estoque
Segundo semestre de Ciência da Computação, um trabalho em web de um controle de estoque. Para o trabalho da A3, feito com o professor Richard. 

# Descrição
Este projeto foi desenvolvido no NetBeans e foi chamado "ControleEstoqueWeb".

# Contextualização
Uma empresa de logística precisa de uma página web para controlar o seu estoque de produtos, e separá-los em categorias. 

# Interface gráfica WEB
As interfaces fazem uso do arquivo "estilo.css" para formatar e melhorar o visual das páginas.
